# Noalle Dashboard

## Overview
Business dashboard for Noalle Jewelry - CRM, social media content creation, and unified analytics.

## Stack
- Next.js 15 (App Router)
- Prisma + SQLite
- Tailwind + shadcn/ui
- Port: 3007

## Key Decisions
- Hybrid approach: Dashboard handles CRM + captions + photo enhancement + analytics
- Posting done through Meta Business Suite (IG/FB) and Pinterest Business (free, no API maintenance)
- Hebrew-first UI (RTL, Heebo font)
- SQLite for zero-config local database
- fal.ai Nano Banana for photo enhancement
- Claude API for Hebrew caption generation

## Brand Identity

- Accent: #e8b4b8 (soft rose)

- Font: Heebo
