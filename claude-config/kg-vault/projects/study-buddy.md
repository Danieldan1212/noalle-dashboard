# Study Buddy

## Overview
University study tool for Economics, Business, Management, and Accounting. Hebrew-first.

## Stack
- Next.js 15 (App Router)
- Prisma + SQLite
- Tailwind + shadcn/ui
- Port: 3008

## Key Decisions
- All UI in Hebrew (RTL)
- Claude API for question generation, concept explanation, study summarization
- Adaptive quiz engine tracks weak topics and adjusts difficulty
- Exam scraper for university portal (Moodle compatible)
- PDF parsing for past exam analysis
- Israeli real-world examples in explanations (Teva, Wix, BoI, TASE)

## Courses
- מיקרו-כלכלה (Microeconomics)
- מאקרו-כלכלה (Macroeconomics)
- חשבונאות פיננסית (Financial Accounting)
- מנהל עסקים (Business Management)
- סטטיסטיקה עסקית (Business Statistics)
