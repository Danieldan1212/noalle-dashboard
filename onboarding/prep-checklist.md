## 2. חשבונות ומפתחות API (Accounts & API Keys)

צור חשבון בכל אחד מהשירותים האלה. **שמור את הסיסמאות!**

### חובה (Required)
- [ ] **Anthropic** → console.anthropic.com
  - הירשם → הוסף אמצעי תשלום → צור API Key
  - זה מריץ את Claude Code עצמו
  - עלות: ~$5-20/חודש לפי שימוש

### לתמונות (Image Enhancement)
- [ ] **fal.ai** → fal.ai
  - הירשם → הוסף אמצעי תשלום
  - שיפור תמונות באמצעות AI
  - עלות: ~$0.01-0.05 לתמונה (זול מאוד)

### לחנות (Shopify)
- [ ] **Shopify Admin API**
  - לך ל: Shopify Admin → Settings → Apps and sales channels
  - לחץ "Develop apps" → "Create an app"
  - תן הרשאות: Products, Customers, Orders, Analytics (קריאה + כתיבה)
  - העתק את ה-Admin API access token

### לרשתות חברתיות (Social Media)
- [ ] **Meta Developer** → developers.facebook.com
  - צור אפליקציה חדשה (Business type)
  - חבר את חשבון Instagram Business + דף Facebook
  - קבל access token עם הרשאות: instagram_basic, instagram_content_publish, pages_manage_posts, pages_read_engagement
  
- [ ] **Pinterest Developer** → developers.pinterest.com
  - צור אפליקציה → קבל access token
  - הרשאות: boards:read, pins:read, pins:write, user_accounts:read
