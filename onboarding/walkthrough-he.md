# 🎯 מדריך התחלה מהירה - Noalle Jewelry + Study Buddy

## מה זה Claude Code?

Claude Code הוא עוזר חכם שיושב על המחשב שלך. אתה כותב לו מה אתה רוצה - בעברית או באנגלית - והוא עושה את זה.

**דוגמה:**
```
אתה: "תפרסם את התמונה הזאת באינסטגרם עם כיתוב בעברית"
Claude: *משפר את התמונה, כותב כיתוב, מוסיף האשטגים, ומפרסם*
```

---

## 🛍️ הדשבורד העסקי (Noalle Dashboard)

### איך מפעילים
1. פתח VS Code
2. פתח את התיקייה `noalle-dashboard`
3. בטרמינל כתוב: `npm run dev`
4. פתח את הדפדפן ב: `http://localhost:3007`

### מה יש שם

#### 📋 ניהול לקוחות (CRM)
לוח קנבאן שמראה את כל הלקוחות שלך לפי שלב:
- **פנייה ראשונה** - מישהו פנה אליך
- **הצעת מחיר** - שלחת הצעה
- **הזמנה מאושרת** - הלקוח אישר
- **בייצור** - עובדים על ההזמנה
- **נשלח** - ההזמנה יצאה
- **מעקב** - בדיקה שהכל בסדר

**גורר כרטיסיות** מעמודה לעמודה כשהסטטוס משתנה.

#### 📸 מנהל רשתות חברתיות
1. מעלה תמונה של תכשיט
2. לוחץ "שפר תמונה" - ה-AI משפר את האיכות
3. מקבל כיתוב אוטומטי בעברית + אנגלית + האשטגים
4. רואה תצוגה מקדימה של איך הפוסט ייראה בכל פלטפורמה
5. בוחר לאן לפרסם (Instagram / Facebook / Pinterest)
6. לוחץ "פרסם" או "תזמן"

#### 📊 אנליטיקס
דשבורד אחד שמראה הכל:
- **מכירות** - הכנסות, הזמנות, מוצרים מובילים
- **רשתות חברתיות** - לייקים, עוקבים, פוסטים מובילים
- **תובנות** - מה עובד, מה לשפר

## 📖 Study Buddy

### איך מפעילים
1. פתח VS Code
2. פתח את התיקייה `study-buddy`
3. בטרמינל כתוב: `npm run dev`
4. פתח את הדפדפן ב: `http://localhost:3008`

### מה יש שם

#### ❓ תרגול (Quiz)
- בוחר קורס (למשל "מיקרו-כלכלה")
- בוחר נושא (למשל "ביקוש והיצע")
- מקבל שאלות אחת אחת
- עונה ומקבל משוב מיידי בעברית
- בסוף רואה ציון + מה לחזור

#### 📄 סדנת מבחנים
- מעלה מבחנים ישנים (PDF)
- ה-AI מנתח את הדפוסים
- יוצר מבחן תרגול חדש באותו סגנון
- אפשר להדפיס או לתרגל על המסך

#### 📝 סיכומים
- מעלה הרצאה או חומר לימוד
- מקבל דף סיכום מרוכז:
  - נקודות מפתח
  - נוסחאות
  - הגדרות
  - טיפים למבחן

#### 💡 הסבר מושגים
- כותב מושג (בעברית או אנגלית)
- מקבל הסבר פשוט בעברית
- עם דוגמאות מהעולם הישראלי

---

## ⚡ פקודות מהירות (ב-Claude Code)

במקום להשתמש בדשבורד, אפשר גם לכתוב פקודות ישירות:

```
/noalle-post ring.jpg "טבעת זהב"     → מפרסם פוסט
/noalle-analytics                     → דוח מהיר
/study-buddy quiz "חשבונאות"          → תרגול
/study-buddy explain "עלות שולית"     → הסבר מושג
```

## 🆘 בעיות נפוצות
